# Guía de Contribución (CONTRIBUTING)

¡Gracias por tu interés en contribuir a este proyecto! Para mantener el orden y la calidad del código, por favor sigue estos pasos:

## Pasos para contribuir
1. Asegúrate de estar en la rama `develop` y actualízala: `git pull origin develop`.
2. Crea tu rama de trabajo siguiendo nuestras convenciones: `git checkout -b feature/tu-nueva-funcionalidad`.
3. Realiza tus cambios y haz commits usando Conventional Commits (ej. `feat: agregar nuevo botón de login`).
4. Sube tu rama al repositorio remoto: `git push origin feature/tu-nueva-funcionalidad`.
5. Abre un Pull Request (PR) apuntando a la rama `develop`.
6. Solicita la revisión de un compañero. Una vez aprobado, el PR será integrado.
